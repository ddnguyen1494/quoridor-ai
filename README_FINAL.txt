CPSC481 Final Project - Quoridor
by: Daniel Domingo, Daniel Nguyen, David Brown, Charles Bucher

How to run executable:
.exe file can be found in the UnityQuoridor->Builds Folder

How to run Project in Unity:
(To open project, Unity v2017.2.0f3 must be downloaded)
(Unity is not required to view scripts)

- Open up Unity and select "Open" at top right.
- Choose whole folder [UnityQuoridor] containing the Quoridor project.
- Project should then open in Unity and can be manipulated.

- Scripts can be found in the UnityQuoridor->Assets Folder
	- There are scripts in the Agent, Board, and Scripts Folders
